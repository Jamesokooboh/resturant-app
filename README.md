# Savory Table

A small Node.js restaurant app starter with:

- a landing page
- a full menu page
- a sign up page
- a sign in page
- a protected dashboard page
- a reservations page for signed-in guests
- local user storage in `data/users.json`
- local reservation storage in `data/reservations.json`

## Run it

1. Make sure Node.js is installed.
2. Open a terminal in this folder.
3. Start the app:

```bash
npm start
```

4. Visit `http://localhost:3000`

## Notes

- This project uses only built-in Node.js modules, so there are no package dependencies to install.
- User accounts are stored locally in `data/users.json`.
- Reservations are stored locally in `data/reservations.json`.
- Sessions are stored in memory and reset when the server restarts.
